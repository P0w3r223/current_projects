# Archive — Legacy Python Exercises

> This branch preserves my earliest Python practice scripts for history.
> They are **not** part of the active portfolio. The curated portfolio lives on the
> [`master`](https://github.com/P0w3r223/current_projects/tree/master) branch.

These were small console exercises written while learning Python fundamentals
(control flow, classes, `input`/`print`, `random`). They are kept here as a record
of where the journey started — not as showcase projects.

| File | What it actually is |
|------|---------------------|
| `game(hangman).py` | Tic-tac-toe game (human vs. random computer) |
| `hangman.py` | Player classes used by the tic-tac-toe game |
| `guessing_game.py` | Guess-the-number (1–10) |
| `rock_paper_scissors.py` | Rock–paper–scissors vs. computer |

For the current data / AI portfolio, see the
[`master` branch README](https://github.com/P0w3r223/current_projects/tree/master).
